Positioning

MCP is the execution layer, not the launch product.

FDE Consultants Protocoles earns trust through the local Skill, Docs, Academy, and market intelligence. The hosted MCP layer should only add value where teams need shared context, governed tools, auditability, and collaboration.

Readiness checklist

What must be true before production claims.

PersistencePostgres-backed durable tenant state.
Rate limitsRedis-backed controls, not process memory.
BillingStripe webhooks idempotent and tested.
AuthTenant-aware authorization and strict CORS.
MonitoringStructured logs, metrics, alerts, and error tracking.
Load testsNon-fake pass/fail under realistic concurrency.
Governance model

Least privilege by default.

MCP tools should expose narrow business actions, not universal shell power. Resources should be scoped to tenant and workflow. Human review should remain required for high-impact or irreversible actions.
